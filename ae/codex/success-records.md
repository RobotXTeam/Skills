# Demo 成功记录

此文件记录每次完整执行 Demo 输出工作流（全部 13 步）并成功完成的 demo。

## 成功标准

**只有完成以下全部 13 步，才算成功，才能写入此文件：**

1. 搜索知识库
2. 规划部署步骤
3. 部署到设备
4. 验证输出
5. 多模态检验（代码输出）
6. 录制视频流和图片
7. 多模态检验（视频/图片）
8. 用户审核通过
9. 生成 Wiki 草稿文档
10. 上传运行包、模型和证据到 Google Drive
11. 推送到 GitHub
12. GitHub 干净克隆验证闭环通过
13. 最终写定 Wiki

**以下情况不算成功，不写入此文件：**
- 用户中途停止（未完成全部 13 步）
- 某一步失败后未继续
- 用户未审核通过
- GitHub 验证闭环未通过
